# Purpose of the programs

Several C ++ classes that I reuse in my projects are stored in these directories.

Directory [inifile](https://github.com/Sunblogger/CPP_Sources/tree/main/inifile): Reading of ini files (text-based). This can be used to read configuration files for programs.

Directory [logfile](https://github.com/Sunblogger/CPP_Sources/tree/main/logfile): Writing of log files to record the status of the program.

Directory [stopfile](https://github.com/Sunblogger/CPP_Sources/tree/main/stopfile): Reading a stop file, the existence of which indicates to a program that it should terminate. A content can also be specified in the stop file, which must be present in order to terminate the program.

Sample programs that show how the classes are used are also stored in the directories.

# Zweck der Programme

In diesen Verzeichnissen sind mehrere C++-Klassen abgelegt, die ich in meinen Projekten wiederverwende.

Verzeichnis [inifile](https://github.com/Sunblogger/CPP_Sources/tree/main/inifile): Lesen von ini-Dateien (textbasiert). Damit können Konfigurationsdateien für Programme gelesen werden.

Verzeichnis [logfile](https://github.com/Sunblogger/CPP_Sources/tree/main/logfile): Schreiben von Log-Dateien, um Status vom Programm festzuhalten.

Verzeichnis [stopfile](https://github.com/Sunblogger/CPP_Sources/tree/main/stopfile): Lesen einer Stop-Datei, deren Existenz einem Programm anzeigt, dass es sich beenden soll. Es kann auch ein Inhalt in der Stop-Datei festgelegt werden, der vorhanden sein muss, um das Programm zu beenden.

In den Verzeichnissen sind auch Beispielprogramme abgelegt, die die Anwendung der Klassen zeigen.
